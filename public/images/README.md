# Imágenes del Portafolio

## Instrucciones para Agregar tu Foto

1. **Nombre del archivo**: `profile.jpg`
2. **Ubicación**: Esta carpeta (`public/images/`)
3. **Tamaño recomendado**: 
   - Mínimo: 300x300px
   - Óptimo: 400x400px o mayor
4. **Formato**: JPG, PNG o WebP
5. **Consideraciones**:
   - Foto clara y profesional
   - Fondo liso o suave
   - Recortada a cuadrado
   - Tamaño archivo < 500KB

## Carpeta de Proyectos
También puedes agregar imágenes de proyectos aquí:
- `tuxtepec.jpg`
- `iepc.jpg`
- `gps.jpg`

Los proyectos buscarán imágenes en `/projects/` dentro de la carpeta `public/`.
